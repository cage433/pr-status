Usage: {script} [-c CONFIG] [-m MARKS]

Options:
  -c CONFIG     Path to config file (default: ~/.conf/pr-status/config)
  -m MARKS      Path to marks file  (default: ~/.cache/pr-status/marks)
  -h            Show this help message

Commands:
  r, report [cols] [options] [--sort cols] [--filter filter-expression]
                       output a report
  m, mark PR           Record current time for PR
                        - Make a note of the current time against the specified PR in MARKS.
                          Comments earlier than the marked time will be ignored by the report
  unmark PR            Remove mark for PR
                        - equivalently, simply delete the corresponding row in MARKS
  f, focus PR          Focus on a specific PR
                        - all reports will be filtered to show only that PR
                        - the prompt changes to #PR>
  u, unfocus           Remove focus
  rl, reload           Reload the config file
  h, help [topic]      Show this help message, or a specific topic
  quit / exit          Exit
  <alias>              Run the report corresponding to this alias, defined in the config file

Command Options:

  --include-ai         Include ai comments (ignored by default)

Sorting:

  --sort col,col,...   
                        - LOC and NC sort descending; all others ascending
                        - 'help columns' lists the available columns
                       

Filtering:

  --filter filter-expression 
                        - keep only rows where the filter-exprssion is true.
                        - multiple --filter flags are ANDed together
                        - 'help filtering' gives more details

Help topics:

  help columns          List available report columns
  help filtering        Filter expression syntax
  help examples         Report examples

Config file structure:
  see sample.config
