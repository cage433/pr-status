Usage: {script} [-c CONFIG] [-m MARKS]

Options:
  -c CONFIG     Path to config file (default: {config})
  -m MARKS      Path to marks file  (default: ~/.cache/pr-status/marks)
  -h            Show this help message

Commands:
  r, report [cols] [options]   Output a PR report
  t, timely [cols] [options]   Output a Timely time-tracking report
  rtc [--num-days=N | --all]   Refresh Timely event cache (default: last 7 days)
  m, mark PR                   Record current time for PR
                                - Comments earlier than the mark will be ignored
  unmark PR                    Remove mark for PR
  f, focus PR                  Focus on a specific PR
                                - All reports are filtered to show only that PR
                                - Prompt changes to #PR>
  u, unfocus                   Remove focus
  rl, reload                   Reload the config file
  h, help [topic]              Show help, or a specific topic
  quit / exit                  Exit
  <alias>                      Run the alias defined in the config file

Report Options:

  --include-ai         Include AI comments and reviewers (excluded by default)
  --include-drafts     Include draft PRs (excluded by default)
  --include-pre-mark-commits
                       Include comments before the mark timestamp

Sorting:

  --sort col,col,...
                        - Sorts ascending by default: alphabetical for text,
                          smallest-first for numbers, oldest-first for dates
                        - Append :R to reverse a column's sort order
                          e.g. --sort NC:R,author sorts by NC descending,
                          then author ascending
                        - 'help columns' lists available column names

Filtering:

  --filter filter-expression
                        - Keep only rows where the filter-expression is true
                        - Multiple --filter flags are ANDed together
                        - 'help filtering' gives more details

Help topics:

  help columns          List available report columns
  help filtering        Filter expression syntax
  help examples         Report examples

Config file structure:
  see sample.config
